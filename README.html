Hello! This is Kishor. This is the readme for my implementation of a lexer engine.

The project structure:

    -> pr01
      -> python-example
        ->bin
          -> pr01(Executable of my program)
        ->activate.sh(source this file for creating environment)
       -> bin
        -> student-grader(program takes the path of my executable and tests against 19 test cases)

To run my program. Just pass a string to be tokenized to stdin, and pipe it to my program, with one argument that is path to the nfa program.

Run this script inside the bin of python-example directory.
echo "string goes here" | ./pr01 "path to the NFA program" You can also run pytest -v in the python-example directory that runs the unit test of my program.
